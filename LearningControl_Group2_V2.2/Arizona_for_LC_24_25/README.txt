RUN_Arizona_base_GOAT.m

You can change settings in the box entitled as:
%% ========================================================================
% ARIZONA OPTIONS - settings that can be changed
% =========================================================================


If you need to do precalculations, that can be done here:
%% ========================================================================
% Student Pre-Build calculations
% =========================================================================

To implement your repetitive controller in Simulink, goto \Runfiles\Simulink and open
Arizona_base_GOAT.slx
There is a cyan subsystem called "repetitive controller", make changes inside this subsystem and do not change other parts of the simulink model.
